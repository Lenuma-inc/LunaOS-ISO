# LunaOS-ISO

These are the basic needed files and folders to build LunaOS system ISO.

## You will find the ISOs here:
https://lenuma.ru/iso/lunaos-latest-x86_64.iso

### Base source

- [Arch-ISO](https://gitlab.archlinux.org/archlinux/archiso)
- [EndeavourOS-ISO](https://github.com/endeavouros-team/EndeavourOS-ISO)
- [CachyOS-Live-ISO](https://github.com/CachyOS/CachyOS-Live-ISO)
- [Calamares](https://gitlab.com/LunaOS/iso/lunaos-calamares)
- [mkarchiso](https://gitlab.com/LunaOS/pkgbuilds/lunaos-core-pkgbuilds/system/base/mkarchiso)
